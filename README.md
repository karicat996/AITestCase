AI测试用例自动化生成文档

1. 基于AI的LLM大模型，生成json数据，转换成测试点
2. 测试点基于模板生成测试用例，测试用例转换为xmind
3. 测试用例支持tapd，禅道


### 基本原理

 1. 输入：  支持文本，pdf文件，md文件，纯图片
 2. 采样器:  喂的模板数据
 3. AI模型选择：需要登录账号
 4. 数据格式转换，数据获取，数据导出
 5. 输出：  json数据，xmind数据
 6. 测试用例生成：
 7. 测试用例模板： 禅道模板，tapd模板
 8. 导出为excel,支持自定义模板
 9. qt框架UI界面


功能1
用户上传图片、文本或者两者
选择范围选项 （web,app,pc,小程序）测试参考指标
选择 测试点生成模板

功能2
测试点生成测试用例excel

功能3
测试用例快速导入tapd，禅道


分支功能
1. 新建和修改测试参考模板
2. 修改测试点模板和测试用例模板
